# matlab-steam-stimulation-model
MATLAB reproduction of a semi-analytical model for reservoir temperature distribution during cyclic steam stimulation (CSS).
This repository contains MATLAB code to reproduce the model described in:

**Thermodynamic analysis and prediction of reservoir temperature distribution for steam stimulation**  
Gang Cheng, Zhanxi Pang, Yanming Jiang, Bo Wang, Xiaogang Yu  
Journal of Petroleum Science and Engineering, 2019  

https://doi.org/10.1016/j.petrol.2019.106394

The paper develops a semi-analytical model for predicting reservoir temperature distribution and the movement of thermal fronts during cyclic steam stimulation (CSS) in a horizontal heavy-oil reservoir.

## Overview

The model considers three radial zones:

1. Steam chamber at steam-saturation temperature
2. Hot liquid zone between the condensation front and thermal front
3. Cold oil zone at initial reservoir temperature

The repository is intended to reproduce the paper’s main results in MATLAB.
